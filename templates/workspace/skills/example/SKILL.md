# Misol Skill

Bu misol skill — faqat namuna sifatida. O'zingizning skill yaratish uchun:

1. `skills/` papkasida yangi papka yarating
2. `SKILL.md` fayl yarating
3. Frontmatter da `name` va `description` yozing
4. Body da agent uchun ko'rsatmalar yozing

Agent bu ko'rsatmalarga amal qiladi — hech qanday kod kerak emas!
